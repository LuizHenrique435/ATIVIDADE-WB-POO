## Execução do projeto

ATIVIDADE I WB

I. No terminal do projeto, na pasta atvi-wb, execute o comando:
```console
    npm install
```
II. Compile o código e gere a pasta out, ainda na pasta atvi-wb:
```console
    tsc
```
III. Rode o arquivo main.js localizado na pasta out:
```console
    node .\out\app\main.js
```
